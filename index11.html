MultiViz

Viking Analytics visualisation tool, MultiViz, helps to understand data and clearly visualize multidimensional anomalies both in historical data as well as in real time. Our proprietary analytics visualisation tool MultiViz can be used for identifying rare events, fingerprinting them and understand the root cause. MultiViz is based on state of the art techniques in machine learning to give the user the most efficient way to handle huge amount of data.
